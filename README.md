# IPL Match Analytics and Winning Insights using Python

## Project Overview

This project analyzes Indian Premier League (IPL) match data using Python, Pandas, Matplotlib, and Seaborn to discover important match-winning patterns and player performances.

The analysis focuses on:
- Toss impact on match results
- Performance during different match phases
- Top batters and bowlers across multiple IPL seasons
- Data-driven cricket insights

---

## Objectives

- Analyze whether teams winning the toss also win matches
- Compare scoring patterns during:
  - Powerplay Overs
  - Middle Overs
  - Death Overs
- Identify the top 5 batters and bowlers
- Visualize insights using charts and graphs

## Key Insights

- Winning the toss does not always guarantee victory.
- Death overs scoring has a stronger connection with winning matches.
- Some players consistently dominate across multiple IPL seasons.
